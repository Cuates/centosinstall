[Install Ruby On Rails CentOS 8](https://www.osradar.com/install-ruby-on-rails-centos-8/)

* **IMPORTANT NOTE**

* `source /etc/profile.d/rvm.sh`

  * **WAIT FOR THIS TO FINISH**
  * Install package 'rubygems' to provide command 'gem'? y
* `rails -v`
